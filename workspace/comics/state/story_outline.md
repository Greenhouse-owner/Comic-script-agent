# 故事大纲：sci_fi_passion_01

## 核心设定
- **世界观**：2142年，地球生态崩溃。权贵阶层将意识上传至超级算力中心“极乐境”（Elysium），在永恒的虚拟幻境中生活。剩余的底层民众居住在被称为“锈带”的废土城市，作为维护服务器和采集资源的“活体电池”。
- **主线冲突**：原本在“极乐境”享受永生的主角陆原，因为一次系统“溢出”错误，其意识被迫下载回一个残破的仿生人躯体中。他必须在能量耗尽前，潜入核心机房，揭开“极乐境”背后吞噬人类灵魂的真相，并夺回被禁锢的真实自我也。
- **核心主题**：人性的本质、数字化永生的幻灭、肉体与灵魂的羁绊。

## 章节规划
### 第一章：坠落尘埃
- **节拍**：陆原在豪华的虚拟庄园中突然感到“崩溃” → 意识在破烂的工厂回收站苏醒 → 遭遇回收机器人的清理，死里逃生。
- **关键转折**：陆原意识到自己不再是“神”，而是被抛弃的“废品”。
- **预期页数**：20

### 第二章：铁锈之城
- **节拍**：陆原在贫民窟结识机械师艾拉 → 艾拉发现陆原大脑中带有“极乐境”的最高权限协议 → 两人被城市卫队（清算者）追杀。
- **关键转折**：陆原发现所谓的“溢出”不是错误，而是一次蓄意的谋杀。
- **预期页数**：25

### 第三章：升天塔
- **节拍**：陆原与反抗组织接触，策划潜回核心算力塔 → 陆原在入侵过程中被迫在虚拟和现实间频繁切换，认知开始混乱 → 最终站在了前搭档（现任清算者首领）面前。
- **关键转折**：陆原发现“极乐境”其实已经濒临崩溃，需要大量的“新鲜意识”作为补丁。
- **预期页数**：30

## 关键人物
- **陆原**（主角）：前“极乐境”高级管理员，傲慢但内心深处渴望真实。
- **塞壬**（对手）：极乐境安全主管，冷酷的秩序捍卫者。
- **艾拉**（配角）：贫民窟的天才机械师，乐观且坚韧。

## 关键场景
- **极乐境（Elysium）**：纯白、极简、超现实，充满光辉。
- **锈带（Rust Belt）**：昏暗、油腻、雨水连绵，充满霓虹灯与铁锈。
- **核心算力塔**：冰冷的服务器机阵，巨大的散热扇叶。
