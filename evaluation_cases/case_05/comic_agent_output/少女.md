# 少女

背景故事未明确，主要表现为温柔、内向、喜静的特质。她似乎一直对身边的事物充满好奇但又害羞，喜欢安静的环境和细腻的情感表达。

外貌特征：
- 柔和的面部线条
- 长发，戴着简单的发饰
- 穿着朴素但舒适的衣服，色调柔和

性格特征：
- 内心温暖、偏执细腻
- 喜欢观察，善于倾听
- 害羞，不善表达

关系：
- 活动范围：静静阅读、写作、独自散步
- 朋友关系：少，有几个理解她的好友

外在环境：
- 常在静谧的自然环境中出现，比如森林、湖边、花园
- 喜欢柔和的光线，偏好晨光或黄昏的景色

情感状态：
- 细腻而深沉，偶尔会表现出淡淡的忧愁
- 喜欢被人理解和陪伴，渴望温暖和安全感
